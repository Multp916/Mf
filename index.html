Hipervinculos

Los hipervinculos nos ayudan para poder transefirnos de una pagina a otra o tambien de una parte del sitio a otra la etiqueta se llama < a href="(link)"> < /a > asi como el link que presionaron en la pagina anterior para direccionarlos aqui.

Ejemplo de la etiqueta

Google

Imagenes

La etiqueta para poner imagenes es la siguiente <"img" src="(imagen descargada y puesta en la carpeta)"> <"/img">

Ejemplo de la etiqueta

Listas

Para el comando de listas podemos variar entre las etiquetas <"ol"> <"/ol"> (listas ordenadas o numeradas) y <"ul"> <"/ul"> (listas sin un orden) y la etiqueta <"li"> para poner un elemento en la lista

Ejemplo de la etiqueta <"ol">

  1. Aries
  2. Tauro
  3. Géminis
  4. Cáncer
  5. Leo
  6. Virgo
  7. Libra
  8. Escorpión
  9. Sagitario
  10. Capricornio
  11. Acuario
  12. Piscis

Ejemplo de la etiqueta <"ul">

  • Uno
  • Dos
  • Tres
  • Cuatro
  • Cinco
  • Seis
  • Siete
  • Ocho
  • Nueve
  • Diez
